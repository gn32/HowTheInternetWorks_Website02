Internet Architecture

Important Terminology-

  • Client – uses web browser to request pages from an internet server

  • LAN – Local Area Network connects computers close to one another

  • WAN – Wide Area Network connects two or more LANS together

  • IXP – Internet Exchange Point to connect WANs together

  • WAN – Wide Area Network connects two or more LANS together

  • ISP – Internet Service Provider owns WANs and leases access to users

    • How an Internet Message is sent-

  • Data being sent is divided into packets (Max size = 64K, 65535 bytes )
  • Packet has a header with some technical information and a FROM address, TO address and the actual data. Each header is 60 bytes
  • Packets divided in to multiple parts contain the order of the packet
  • Packets move on the Internet along different paths each taking different times
  • Packets are reordered
  • Received packets are acknowledged back to the sender
  • Sender resends packets that do not acknowledge
